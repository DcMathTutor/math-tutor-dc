This project contains a basic website hosted on GitHub Pages for a tutoring business. 

The goal of the business is to expand to 200k monthtly revenue.

This business functions by matching recently graduated college students with highly qualified math and statistics backgrounds with early college/high school tutees.
We collect a small margin for sourcing clients, connecting the two, collecting payments, scheduleing, and providing tutoring materials. 

The website shows a form to sign-up, which automatically sends an email via resend
through a backend hosted on Render(see Desktop/backend-tutor)

All scheduling and payment collection is done manually.
